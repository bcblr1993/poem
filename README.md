# poem
唐诗检索系统

# 技术选型
- springboot
- redis
- es
- mysql
- kibana
- mybatis
- jsp


# 数据库
- poem.sql

# 安装
- 创建数据库 poem
- 导入poem.sql文件
- 安装docker、docker-compose
- 使用docker-compose启动服务
- idea导入项目并修改配置信息为自己数据库配置信息以及服务相关配置
- 启动项目
- 访问：
    <br/>
    
    ``前台地址：http://localhost:8989/poem/front/index.jsp``
    ``后台地址：http://localhost:8989/poem/back/index.jsp``
